# FastMD5

A fast, threaded, batch MD5 solver
Can be executed with args (e.g. FastMD5.exe <xmlfile> e.g. FastMD5.exe input.xml) or through the GUI.

Config file example:
<?xml version="1.0"?>
<entries>
	<entry>
		<name>bin1.txt</name>
		<hash>C8A1D65AAB21EBEDE5D6983671FF2307</hash>
	</entry>
	<entry>
		<name>bin2.txt</name>
		<hash>C8A1D65AAB21EBEDE5D6983671FF2307</hash>
	</entry>
	<entry>
		<name>bin3.txt</name>
		<hash>C8A1D65AAB21EBEDE5D6983671FF2307</hash>
	</entry>
	<entry>
		<name>bin4.txt</name>
		<hash>C8A1D65AAB21EBEDE5D6983671FF2307</hash>
	</entry>
</entries>
